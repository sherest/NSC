A – Appearance

0 = cyanotic or pale all over
1 = normal/pink body core, blue extremities
2 = normal/pink core and extremities

P – Pulse

0 = none
1 = pulse <100
2 = pulse >100

G – Grimace (flick soles, observe face)

0 = no reflex activity
1 = only facial grimace
2 = grimace and cough/sneeze/cry

A – Activity

0 = limp, no movement
1 = flexion but no active movement
2 = active movement

R – Respiratory Effort

0 = none
1 = slow/irregular breathing, weak cry
2 = good breathing, strong cry

Scoring:

7–10 Provide routine care
4–6 Provide oxygen and stimulation
0–3 Provide oxygen and CPR
